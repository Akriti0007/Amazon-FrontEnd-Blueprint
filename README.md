# Prime-Flex-Layout (Amazon UI Clone)

A high-fidelity front-end implementation of a global e-commerce landing page. This project focuses on mastering complex CSS layouts, responsive design patterns, and semantic HTML structure.

## 🛠️ Tech Stack
* **HTML5**: Semantic structure for accessibility and SEO.
* **CSS3**: Advanced Flexbox layouts, hover transitions, and asset positioning.
* **FontAwesome**: Scalable vector icons for UI elements.

## 🧠 Key Learning Objectives
* **Complex Navigation Systems**: Replicated a multi-layered navbar including search dropdowns, location selectors, and account management links.
* **Flexbox Architecture**: Utilized `flex-wrap: wrap` and `justify-content: space-evenly` to create a dynamic, multi-column product grid.
* **Asset Integration**: Managed various image formats and background-size properties to maintain visual consistency across different "Shop Section" categories.
* **Deep-Footer Design**: Built a comprehensive four-panel footer to practice organizational hierarchy and nested list styling.

## 📂 Project Structure
* `index.html`: Core structure and content.
* `style.css`: Custom styling and layout logic.
* `/assets`: Visual assets including category images and logos.

## 🎨 Features
* **Interactive Borders**: Smooth hover states on navigation elements for enhanced UX.
* **Global Hero Section**: Large-scale hero imagery with a localized messaging overlay.
* **Responsive Grid**: A 4-column product display that adjusts based on container width.
